## VGC College Management System

How to Run the Project Locally

### **Prerequisites:**

.NET 8 SDK (or higher) installed.

**Visual Studio 2022, JetBrains Rider, or VS Code.**

**dotnet tool install --global dotnet-ef**

### Steps:

Clone the repository to your local machine.

Open the solution (.sln) in your IDE.

Open the Terminal and run the command to apply database migrations (SQLite):

**dotnet ef database update**

Press F5 or click Run to start the application. The browser will automatically open at https://localhost:7xxx.
